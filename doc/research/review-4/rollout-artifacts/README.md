# Rollout Artifacts

Store per-task rollout proof artifacts here.

Recommended structure:

- `doc/research/review-4/rollout-artifacts/<phase>/<task>/before/`
- `doc/research/review-4/rollout-artifacts/<phase>/<task>/after/`
- `doc/research/review-4/rollout-artifacts/<phase>/<task>/validation/`

Each completed task should link its artifact paths in:

- `doc/research/review-4/ROLLOUT_EXECUTION_PLAN.md` (Change Journal)
- `doc/research/review-4/ROLLOUT_EXECUTION_PLAN.md` (Issue Closure Register)
